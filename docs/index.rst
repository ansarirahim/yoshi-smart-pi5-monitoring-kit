Overview
--------

The Raspberry Pi Smart Monitoring Kit is a comprehensive solution for adding intelligent
monitoring capabilities to Wi-Fi cameras using Raspberry Pi 4.

Features
--------

* Motion and fall detection using OpenCV
* LINE Messaging API integration for notifications
* OTA (Over-The-Air) updates from GitHub
* Pan-tilt camera control with auto-tracking
* Voice alerts for anomaly detection
* 100% local processing (no cloud dependency)
* Privacy-first architecture
